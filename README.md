# SkillTracker
**Stack: HTML · CSS · JavaScript · Supabase (backend + database + auth)**

---

> SkillTracker is a polished learning dashboard that helps users build consistency, log progress, and visualize skill growth with a beautiful web experience.

---

## 💡 What SkillTracker Does

SkillTracker is a complete self-learning companion with:
- secure Gmail-only login
- personal skill management
- session logging for every practice moment
- progress charts that turn effort into insight
- an anonymous community board for sharing ideas

It is built with **HTML, CSS, JavaScript**, and **Supabase** for backend authentication and database storage.

---

## ✨ Features

- **Gmail-only sign-up and login**
- **Strong password validation** with live rules check
- **Persistent Supabase sessions** across refreshes
- **Add, edit, and delete skills** easily
- **Set goals for each skill** you want to build
- **Log practice sessions** with notes and timestamps
- **Beautiful progress charts** powered by Chart.js
- **Inactivity alerts** for skills not practiced in 30+ days
- **Anonymous community posts** for sharing tips
- **Responsive UI** for desktop and mobile
- **Smooth animations** and modern landing page design
- **Cloud-powered data storage** using Supabase
